Bookord is a real-time chat application.

I used this project to learn how to work with Firestore, a database system provided by Firebase. App also supports the Firebase Authentication system.

The data is updated in real time and React handles rerendering necessary components. States are managed by Context API.

For styling, I worked with Bootstrap React components.

Demo is setup on free account, so the API calls are limited.

The project was deployed to Firebases' Hosting.
